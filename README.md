This project analyzes customer behavior and estimates Customer Lifetime Value (CLV) using Python, SQL, and Power BI. 
The objective is to identify high-value customers, measure retention probability, segment customers based on business value, and recommend targeted retention strategies.

The analysis was performed on a dataset containing 53,503 customer records.
